description: Show status of all agentic-config installations
---

Query agentic-config installations using the agentic-status agent.

The agent will:
1. Read central installation registry
2. Check each project's current version
3. Validate symlink integrity
4. Show comprehensive status report
5. Highlight outdated or broken installations
6. Provide remediation commands

Use this to get an overview of all projects using agentic-config and their health status.
